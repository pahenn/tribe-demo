# King James Bible

The complete King James Version (KJV) of the Holy Bible, built as a browsable documentation site with [Docus](https://docus.dev) and [Nuxt Content](https://content.nuxt.com/).

## Structure

```
content/
├── index.md               # Landing page
├── 1.old-testament/       # 39 books (Genesis–Malachi)
└── 2.new-testament/       # 27 books (Matthew–Revelation)
```

## Development

```bash
pnpm install
pnpm dev
```
